Phishing Awareness System
About the Project

This project is a simple Phishing Awareness Simulation built to demonstrate how phishing attacks work in real life.
It helps users understand how fake emails can trick people into entering their credentials and how such activities can be monitored.

The goal of this project is educational — to create awareness about phishing and encourage safer online behavior.

How It Works
1.The user opens a simulated email inbox
2.A fake email is displayed (looks like a real message)
3.When the user clicks the email, they are redirected to a login page
4.If credentials are entered, they are captured (for demonstration only)
5.All entries are displayed in a dashboard with analytics

Technologies Used
Backend: Flask (Python)
Frontend: HTML, CSS
Database: SQLite
Charts: Chart.js
Deployment: Render

Features

Realistic email simulation
Login page to capture inputs (demo purpose)
Dashboard showing:
  Username
  Password (for demo — not recommended in real apps)
  Time of submission
  IP address
Graph showing login attempts over time
Clean and responsive UI

Ethical Note
This project is created strictly for educational purposes only.
It does NOT promote phishing or malicious activity
It is designed to demonstrate how such attacks happen
Users should never enter real credentials
In real-world applications, passwords must be securely hashed and protected

Conclusion
This project helps in understanding:
How phishing emails are designed
How users can unknowingly share sensitive information
The importance of cybersecurity awareness
